# Calculator (PyQt5)

This project is a beautifully styled desktop calculator app made with Python and PyQt5.

---

## ✨ Features

✅ Basic arithmetic operations: +, -, ×, ÷  
✅ Percentage (%) operator  
✅ Positive/negative toggle (+/-)  
✅ Clear (`C`) and error handling  
✅ Responsive, scalable layout

---

## 🛠 Technologies Used

- Python 3.12  
- PyQt5
